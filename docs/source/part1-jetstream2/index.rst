Part 1 — Data Exploration & Preprocessing on Jetstream2
=======================================================

In this part you'll use a Jetstream2 cloud VM as an interactive workbench to
explore the dataset and build the preprocessing pipeline that feeds model
training in :doc:`Part 2 <../part2-anvil/index>`.

.. admonition:: Goals for this section
   :class: tip

   - Launch and connect to a Jetstream2 instance.
   - Set up a reproducible Python/ML environment.
   - Explore the raw dataset interactively.
   - Build and validate a preprocessing pipeline.
   - Stage the processed data for transfer to Anvil.
